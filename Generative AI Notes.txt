4. Different Ways Of creating Python Environment
Using venv (built-in)
python -m venv venv

✔ Default Python tool
✔ Lightweight

Using conda
conda create -p venv python=3.12

✔ Good for Data Science / AI
✔ Manages Python + packages

Using virtualenv
pip install virtualenv
virtualenv venv

✔ Older alternative to venv
✔ More flexible

Using pipenv
pip install pipenv
pipenv install

✔ Combines env + dependency management
✔ Uses Pipfile
